설치만 해 두면, 장치 목록과 Wi‑Fi 도구를 대시보드에서 만납니다.

Scapy·ARP 스캔, 포트·OUI, 로그 가공까지 — 이 사이트는 프로젝트를 소개하는 웹 페이지이고, 실제 패널은 PC에서 서버를 켠 뒤 같은 Wi‑Fi의 다른 기기에서도 열 수 있습니다.

다른 PC·휴대폰에서도 이 페이지를 열 수 있어요

Portfolio와 같이 인터넷만 연결되면 어디서나 브라우저로 접속할 수 있습니다. 별도 프로그램 설치 없이 주소만 입력하면 됩니다.

https://karin01.github.io/NetWork-IP-Search/

반면 실시간 스캔 대시보드(그래프·장치 목록)는 Python 서버가 돌아가는 PC가 있어야 합니다. 그 PC에서 실행한 뒤 http://그PC의IP:5000 으로 같은 Wi‑Fi 기기에서 테스트하세요.

무엇을 하나요

  • LAN 장치 — 스캔 결과를 대시보드에서 정리합니다.
  • Wi‑Fi — 전용 경로에서 netsh·주변 AP·채널 등 (Windows).
  • 문서 — API·로그 랩은 Docs에서 확인합니다.

대시보드 실행 (Windows)

  1. 저장소를 클론하거나 ZIP으로 받습니다.
  2. run_dashboard.bat 또는 대시보드 열기.bat 실행.
  3. 이 PC에서 http://127.0.0.1:5000 — Wi‑Fi 전용은 /wifi.
  4. 다른 기기에서는 같은 망의 http://192.168.x.x:5000 (방화벽 허용 필요할 수 있음).

GitHub Pages는 정적 파일만 호스팅합니다. 그래서 이 웹 소개는 전 세계 어디서나 열리고, 앱 본체는 실행 중인 PC를 기준으로 동작합니다.
