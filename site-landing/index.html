Windows에서 돌아가는 네트워크 허브예요. IP·ARP·간단한 Wi‑Fi 도구까지 — 브라우저로 열어 쓰는 개인용 패널입니다.

이 주소(GitHub Pages)는 웹 소개 페이지입니다. 그래프와 실시간 스캔은 PC에서 서버를 켠 뒤 접속해 주세요.

무엇을 하나요

  • LAN 장치 — 스캔 결과를 대시보드에서 정리해 봅니다.
  • Wi‑Fi — 전용 화면에서 netsh·주변 AP·채널 등을 다룹니다 (Windows).
  • 문서 — API·로그 랩·자동화는 문서 사이트에서 확인합니다.

실시간 대시보드 여는 법

  1. GitHub에서 저장소를 클론하거나 ZIP으로 받습니다.
  2. 폴더에서 run_dashboard.bat 또는 대시보드 열기.bat을 실행합니다.
  3. 브라우저에서 http://127.0.0.1:5000 을 엽니다.
  4. Wi‑Fi 전용은 http://127.0.0.1:5000/wifi 입니다.

GitHub Pages는 정적 호스팅만 가능해 Python 서버를 띄울 수 없습니다. 그래서 Portfolio처럼 “웹사이트”는 여기, 앱 본체는 로컬에서 돌아가는 구조입니다.
